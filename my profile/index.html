LIL'Reader

Une solution permettant d'automatiser le renseignement des cartes de mutuelles dans les officines

La solution que vous attendiez tous, enfin dans vos officines

  • Gain de temps pour les patients derrière le comptoir
  • Gain de temps pour les pharmaciens. Un passage au scanner et le tour est joué
  • Réduction des erreurs
  • Réduction des rejets, donc moins de perte d'argent pour les pharmacies

Solution permise grâce à l'OCR, reconnaissance optique de caractères, et au Machine Learning

L'OCR permet de récupérer les informations dans la carte mutuelle et de le sauvegarder dans un fichier pouvant être exploité dans le logiciel de management de la pharmacie, et stocké dans une base de données ou sur un autre support exploitable par un système informatique
